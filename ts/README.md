# edgartools-ts

This directory holds an experimental TypeScript port of some utilities from the
Python [edgartools](../README.md) project. It currently provides:

- `DataPager` for paginating arrays
- `moneyfmt` for formatting currency values
- A few helper modules (`codes`, `rich`, `markdown`) with simplified
  functionality compared to their Python counterparts
- Basic helpers in `core` such as `Result`, `reverseName`, and `datefmt`

```
npm install
npm run build
```
